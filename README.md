## OM-Packet-Sniffer

This app is python based packet sniffing one (no modification of data)
that includes several other features to help people manage sect event in the game as well as player progression

### The features included are:
- sect duels matchmaking table based on an algorythm that determine the best matchups with contraints such as:
    - BR of the sect players as well as opponents sect players
    - people having dual path (not all realms yet included)
    - number of maximum fights and defeat (3 for each)
- automatic save to an excel files of the matchup table as well as the duel battles
- automatic save to an CSV file of the sect players data for each sect scanned
- viewing tab to review scanned data
- viewing tab to review matchmaking before saving it

### The features yet to be included are:
- automatic interface swapping for the sniffing with possibilty for manual select
